Built on the Google Ads API

Smart Ads AI uses the Google Ads API to create and manage campaigns on behalf of Shopify merchants with their explicit consent.

🔧 What We Do With the API

  • Create Search and Performance Max campaigns
  • Set budgets, bidding strategies, and targeting
  • Generate ad groups with AI-optimized keywords
  • Create responsive search ads with AI-generated copy
  • Read campaign performance metrics
  • Pause, enable, or remove campaigns at merchant request

🔒 Data Handling & Security

  • OAuth 2.0 for secure Google Ads authentication
  • Campaign data accessed only when merchant uses the app
  • No Google Ads data shared with third parties
  • Merchants can disconnect at any time, revoking all access
  • All data transmitted over encrypted HTTPS (TLS 1.2+)
  • Compliant with Google Ads API Terms of Service

🛡️ Merchant Control & Account Ownership

  • Each merchant connects their own Google Ads account via OAuth 2.0 — we never use a shared or intermediary advertising account
  • All campaigns are created in PAUSED state and require explicit merchant approval before any ad goes live
  • Merchants retain full control over their Google Ads account, billing, and campaign settings at all times
  • Merchants can review, edit, pause, or remove any campaign directly in Google Ads or through our dashboard
  • Disconnecting the app immediately revokes all API access to the merchant's Google Ads account
  • We never store Google passwords or billing information
