# Bamzal Brand Mark Options

Exploration set for the standalone Bamzal mark.

- `a-prism.svg` — angular B with internal prism/play cut.
- `d-lift.svg` — B with upward growth arrow.
- `e-gateway.svg` — B with gateway/control-panel insert.
- `h-signal.svg` — B with performance signal bars.
- `compare.html` — local comparison sheet for reviewing all four marks.

Current live site usage:

- Header wordmark: `bamzal-logo.svg` / live HTML wordmark in the navigation.
- Light wordmark: `bamzal-logo-light.svg`.
- Browser/app icon: `favicon.svg`, `favicon.ico`, `favicon-32.png`, `favicon-180.png`, `favicon-512.png`, `apple-touch-icon.png`.
